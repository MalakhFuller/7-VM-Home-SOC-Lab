# 7-VM-Home-SOC-Lab
7-VM home SOC lab on one 64GB host with dual SIEM (Wazuh + Splunk), passive Suricata IDS feeding both, and a hand-rolled CIM mapping into Splunk's Intrusion Detection data model. Documented honestly, dead ends and all, with the troubleshooting kept in. Foundation for the AD attack-simulation project  to follow.
